# Project Configuration

## Markdown Reference Path

All markdown documents for this project are organized under:

```
docs/Sailor/markdown/
```

### Folder Structure

- `notes/` - Working notes, meeting logs, daily observations
- `drafts/` - Draft documents under active writing
- `references/` - Reference materials, specs, external docs
- `templates/` - Reusable document templates

### Usage Rules

- When generating or locating markdown content, default to `docs/Sailor/markdown/` as the base path
- Place new notes in `notes/`, drafts in `drafts/`, reference materials in `references/`
- Use templates from `templates/` when creating structured documents
- Keep folder structure flat within each subfolder unless complexity warrants nesting
